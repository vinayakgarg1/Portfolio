# Personal Portfolio Website

A modern, fully responsive personal portfolio designed to showcase my journey as a software developer, my technical skills, and my recent projects.

🌍 **Live Demo:** [Click here to view my portfolio](https://vinayakgarg1.github.io/HexSoftwares_Portfolio/)

## 📖 About This Project
This website serves as my digital resume. It was built from scratch to demonstrate my foundational web development skills and to provide a central hub for my work, including machine learning applications and web projects.

## 🚀 Features
* **Responsive Design:** Optimized for both desktop and mobile viewing.
* **Project Showcase:** Highlights my recent work with descriptions and visual previews.
* **Downloadable Resume:** Direct link to grab a PDF copy of my current CV.
* **Clean UI:** Built with modern CSS practices like Flexbox and Grid.

## 🛠️ Built With
* HTML5
* CSS3

## 💻 How to Run Locally
To view this project on your own computer:
1. Clone this repository: `git clone https://github.com/vinayakgarg1/HexSoftwares_Portfolio.git`
2. Open the project folder.
3. Double-click the `index.html` file to open it in your default web browser.
