# Modern 3D Developer Portfolio

A high-performance, visually stunning developer portfolio built with **Next.js**, **Tailwind CSS**, and **Three.js**. This project features a dynamic particle network background and a sleek, responsive interface designed to showcase professional work.

## 🚀 Features

* **3D Particle Background:** Interactive particle system built with `react-three-fiber`.
* **Modern UI/UX:** Clean, dark-themed aesthetic with smooth scrolling and glassmorphism.
* **Fully Responsive:** Optimized for all screen sizes from mobile to ultra-wide monitors.
* **Next.js 14:** Utilizing the latest App Router and server-side optimizations.
* **Type Safe:** Built with TypeScript for robust, error-free development.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **3D Graphics:** [Three.js](https://threejs.org/) / [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Deployment:** [Vercel](https://vercel.com/)

### Prerequisites
